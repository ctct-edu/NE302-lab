# 在庫管理システム 要件定義

## 概要

小規模文房具店向けの在庫管理システム。
商品の在庫を管理し、在庫不足時にアラートを表示、発注を支援する。

## ユースケース

### Must（必須機能）

| ID   | 機能         | 説明                               |
| ---- | ------------ | ---------------------------------- |
| UC-1 | 商品の登録   | 新しい商品をシステムに登録する     |
| UC-2 | 商品の編集   | 既存商品の情報を更新する           |
| UC-3 | 商品の削除   | 商品をシステムから削除する         |
| UC-4 | 商品の検索   | 商品名で商品を検索する             |
| UC-5 | 在庫数の確認 | 商品ごとの在庫数を確認する         |
| UC-6 | 入庫の登録   | 商品の入庫を記録し、在庫数を増やす |
| UC-7 | 出庫の登録   | 商品の出庫を記録し、在庫数を減らす |
| UC-8 | 在庫アラート | 在庫が閾値以下の商品を警告表示     |

### Want（あれば良い機能）

| ID    | 機能     | 説明                         |
| ----- | -------- | ---------------------------- |
| UC-9  | 発注作成 | 発注データを作成する         |
| UC-10 | 発注一覧 | 発注状況を一覧表示する       |
| UC-11 | 入荷処理 | 発注した商品の入荷を処理する |

## 技術スタック

| 項目           | 技術                    |
| -------------- | ----------------------- |
| 言語           | Java 21                 |
| サーバー       | Spring Boot 3.x         |
| ORM            | Spring Data JPA         |
| データベース   | MySQL                   |
| フロントエンド | HTML / CSS / JavaScript |
| ユニットテスト | JUnit 5                 |
| E2E テスト     | Playwright              |

## 画面構成

1. **商品一覧画面**

   - 商品の一覧表示
   - 検索機能
   - 商品登録へのリンク

2. **在庫管理画面**

   - 在庫数の確認
   - 入庫・出庫の登録

3. **発注管理画面**

   - 発注一覧
   - 発注の作成・更新

4. **アラート表示**
   - 在庫が閾値以下の商品を警告表示

## ディレクトリ構成

```txt
inventory-system/
├── src/main/java/com/example/inventory/
│   ├── controller/     # REST Controller 定義
│   ├── service/        # ビジネスロジック
│   ├── repository/     # データアクセス（JPA）
│   ├── entity/         # JPA エンティティ
│   └── dto/            # データ転送オブジェクト
├── src/main/resources/
│   ├── static/         # フロントエンド
│   │   ├── index.html
│   │   ├── css/
│   │   └── js/
│   └── application.yml # 設定ファイル
└── src/test/java/
    ├── service/        # JUnit 5
    └── e2e/            # Playwright
```

## 非機能要件

- レスポンスタイム: 1 秒以内
- 同時接続: 10 ユーザー程度を想定
- データ量: 商品数 1,000 件程度を想定
